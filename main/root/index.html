    About Mystery Path

    This project is an interactive story game developed for the SET08101 Web Technologies module.

    It demonstrates DOM manipulation, local storage, responsive design, dynamic content rendering, and JavaScript-based story logic without depending on a framework.

    • Branching narrative structure with multiple endings
    • Inventory tracking and conditional choices
    • Save and resume using localStorage
    • Responsive single-page interface using HTML, CSS, and JavaScript

    How to Play

    • Read the story text carefully before making a choice.
    • Some choices unlock items that affect later scenes.
    • Your run is saved automatically so you can continue later.
    • Try different paths to discover every ending.
